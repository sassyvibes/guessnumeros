# 🎮 Number Guessing Game (Python & C++)

A simple beginner-friendly number guessing game implemented in both Python and C++.

The computer randomly selects a number between 1 and 100, and the player must guess it. The program gives hints whether the guess is too high or too low until the correct number is guessed.

---

## 📂 Files

- `guess_game.py` → Python version
- `guess_game.cpp` → C++ version

---

## 🐍 Run Python Version

Make sure you have Python installed.

```bash
python guess_game.py
